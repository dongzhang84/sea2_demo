# 样板切片：Snr1 / 约翰·法雷尔开场段

这份文档是一个样板，不是总报告。

它只回答一件事：**如果以后再做一条线，我们要怎么把它从原文一路拆到拓扑和设计结论。**

## 1. 为什么选这段

这段是目前最适合做样板的切片，因为它同时包含：

- 身份变化
- 资金筹资
- 船只准备
- 远方目标
- 桥人物交叉
- 后续扩线入口

换句话说，它不是单纯开场，而是整部游戏最典型的“从局部世界进入全局世界”的样本。

## 3. 原始流程

这段剧情可以先压成一条最短流程：

1. 被逐出家门。
2. 去公爵府筹资。
3. 通过公爵夫人、洛克、路琪亞和酒馆关系把出海条件补齐。
4. 造船完成，正式出海。
5. 任务从“活下去”扩展到“找约翰·法雷尔”。
6. 再被推向日本、卡特琳娜和王室阴谋。

## 4. 事件拆分

这一段至少可以拆成这些事件节点：

- 被逐出家门
- 公爵府筹资
- 造船完成
- 恩里克神父同行
- 约翰传闻
- 卡特琳娜卷入

每个节点都应该继续写清楚：

- 触发人物
- 触发地点
- 前置条件
- 状态变化
- 下一跳

## 5. 状态变化

这段最重要的状态变化是：

- 从“家内身份”变成“外部行动者”
- 从“没船”变成“可出海”
- 从“个人生存”变成“世界目标”
- 从“单线成长”变成“跨线交叉”

这说明剧情不是线性讲述，而是状态机推进。

## 6. 设计结论

这一切背后的设计智慧是：

- 开场先压住身份。
- 再用资金和人际关系把玩家推到世界里。
- 再用远方目标把局部冒险变成全局探索。
- 再用桥人物把别的线提前埋进来。

这就是老游戏特别擅长的“短时间内把世界做大”的方法。
