## 一、项目是什么

逆向 KOEI《大航海時代II》(1993, PC-9801, 繁体中文版) 的资源文件，
并用提取出的**素材 + 机制数据**做一个**原创的「大航海」风格游戏**（Godot 4）。

**不是**复刻原作、不是搬运资源、不是做翻译——原作只作为风格蓝本。

源文件不在 git 里，本地路径 `/Users/dong/Projects/Koukai2/`。

## 二、资源逆向成果（已完成）

### 2.1 压缩格式破解

- **LS10 / LS11 / Ls12 解压器**（`scripts/ls11_decode.py`）——KOEI 自有的位流前缀码压缩，
  三种 magic 同一算法。这是所有 `.lzw` 资源的入口。
- 关键经验：盲试参数浪费大量时间，先找现成解码器（参考 tzengyuxio、JohanLi 的 repo）
  比对位流细节快几十倍。

## 三、原创游戏 MVP 成果（已完成）

游戏在 [`../game/`](../game)，Godot 4 项目。**核心玩法闭环已完整可玩**：
低价买货 → 起航 → 途中事件/海战 → 抵达高价卖出赚差价。

已实现功能：

- 世界地图 + 100 个港口（校准投影定位在海岸线上）
- 贸易系统：买卖、12 种商品、区域差价、季节价格波动
- 船只导航 + 风向洋流影响船速
- 航行随机事件（8 种模板）+ 海战
- 港口买船（25 种船型，船坞）
- 存档 / 读档
- 时间 / 日历系统（1502 年起，航行消耗天数）
- 港口氛围：码头 NPC 走动 + 酒馆（打听商情）
- 中文 UI（CJK 字体子集）
- 原创合成 BGM + 音频系统

启动方式：Finder 双击根目录 `play.command`，或在 Godot 编辑器按 F5。

详细路线图见 [`../ROADMAP.md`](../ROADMAP.md)。

## 四、当前卡点（未完成）

### 4.1 事件 CG 图片 —— 没破

`Opgraph.lzw`（~14 张）+ `Graph.dat`（65 张）+ `Endgrp.dat`（59 张结局图），
共约 130+ 张事件 CG。内层是自定义 4bpp + 0x38-escape 压缩，**没破**。

- `Main.exe` 里的解码器入口已定位（0x5e00），结构搞清楚：
  4bpp packed + 0x38 escape + per-image 16-entry 派发表 + 4-plane VRAM 直写。
- 详细反汇编笔记在 [`REVERSE_ENGINEERING_NOTES.md`](REVERSE_ENGINEERING_NOTES.md)，
  完整 disasm 在 [`main_exe_decoder_disasm.txt`](main_exe_decoder_disasm.txt)。
- **Python 移植还没做** —— 这就是下一步攻坚的目标。

### 4.2 事件触发逻辑 + 对话文本 —— 没破

`Event*.dat` / `Snr*.dat`（剧本/触发表）/ `Message.dat`（对话）。
结构识别了，但内容卡在自定义字体/文本编码没解。

### 4.3 6 个主人公

原作有 6 个可选主人公（贸易 João、海盗 Catalina、探险 Ernst、
投资 Pietro、私掠 Ali、冒险 Otto），决斗与走路 sprite 都解出来了，
但**每个主人公的专属事件链和触发条件没有提取**——依赖 4.1 + 4.2。
